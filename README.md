# TaskFlow - To-Do List Web Application

TaskFlow is a simple and modern To-Do List web application built using **HTML, CSS, and JavaScript**.  
This project allows users to **add, edit, and delete tasks** and stores the tasks locally using **Local Storage**. The interface is designed to be clean, responsive, and professional.


##  Features

- Add new tasks
- Edit existing tasks using a modal popup
- Delete tasks
- Tasks are saved in **Local Storage**
- Responsive and modern UI design
- Smooth and professional user experience


##  Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **Local Storage**
